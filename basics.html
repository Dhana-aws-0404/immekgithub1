What is Git?

Git is a version control system that tracks changes in files and helps developers work together.

What is GitHub?

GitHub is a cloud platform that hosts Git repositories and allows collaboration, issue tracking, and project management.

Key Concepts

  • Repository – Project storage
  • Commit – Saved change
  • Branch – Separate development line
  • Pull Request – Merge request
